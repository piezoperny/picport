# picport

# marco's gallery

personal photography portfolio. custom-built for visual focus.

**live:** [pics.pernazza.com](https://pics.pernazza.com)

## features
* **minimalist ui:** grey-on-grey theme, typography-led.
* **responsive:** custom hamburger menu, swipe carousel, centered flex grid.
* **interactive:** spinning logo, lightbox, smooth scrolling.
* **automated:** python script + github actions auto-update the gallery from folders.

## tech
* html5, css3, vanilla javascript.
* python (automation), github pages (hosting).

## workflow
1.  upload photos to `images/MASTER`.
2.  push to github.
3.  github action builds `gallery.json` automatically.
